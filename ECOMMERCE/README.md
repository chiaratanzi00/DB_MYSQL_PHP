#  Ecommerce PHP

Applicazione PHP per gestire una **rubrica contatti** e i loro **ordini**.  
Permette di **aggiungere**, **visualizzare**, **modificare** ed **eliminare** contatti e ordini.  

I dati vengono salvati in **MySQL** tramite **phpMyAdmin/XAMPP**.
